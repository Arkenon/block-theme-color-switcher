This plugin allows users to choose a color palette for the Block Theme from the frontend.

== Description ==

Elevate your WordPress website's user experience with the Block Theme Color Switcher plugin. This powerful yet lightweight tool empowers your site visitors to personalize their browsing experience by selecting from various color palettes directly from the frontend. Compatible with all block themes, it seamlessly integrates with your existing design while adding a layer of customization that users love. The plugin features an elegant, non-intrusive interface that can be positioned exactly where you want it on your site. Transform your static website into an interactive, user-centric experience with just a few clicks - no coding knowledge required. Give your audience the freedom to view your content in a style that suits their preferences, enhancing accessibility and engagement.
